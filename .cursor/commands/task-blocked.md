# /task-blocked

Oznacz task jako zablokowany i przygotuj plan odblokowania.

## Input
- `TASK_ID`
- opis blokera (objaw / kontekst)

## Zrób
1. Ustaw status taska na `BLOCKED` w `tasks.md`.
2. Dopisz wpis `BLOCKER` do `log.md`.
3. Wygeneruj sekcję „Plan odblokowania”:
   - hipotezy (2–4)
   - testy diagnostyczne
   - expected result
   - kolejność działań
4. Zaproponuj:
   - `/troubleshoot-capture` (jeśli problem techniczny)
   - `Next step (1)` na 15–30 min

## Ważne
- Nie przeskakuj od razu do generowania kodu/poprawek.
- Najpierw diagnoza.
